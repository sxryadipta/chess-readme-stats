# ♟️ Chess README Stats

> Generate beautiful, dynamic Chess.com profile cards and last-match summaries for your GitHub README, portfolio, blog, or website.

[![Live Demo](https://img.shields.io/badge/Live-Demo-success)](https://chess-readme-stats.vercel.app/)
[![Chess.com](https://img.shields.io/badge/Powered%20by-Chess.com-blue)](https://www.chess.com/)
[![SVG](https://img.shields.io/badge/Output-SVG-orange)](#)
[![Open Source](https://img.shields.io/badge/Open%20Source-Yes-brightgreen)](#)

Turn your Chess.com activity into shareable SVG cards that automatically update whenever your rating, games, or recent matches change.

## ✨ Features (to be included)

- 📈 Dynamic Chess.com profile statistics
- ♟️ Last match summary card
- 🎨 SVG output for GitHub READMEs, websites, and portfolios
- ⚡ Real-time data fetched from Chess.com
- 📱 Responsive card design
- 🚀 Zero setup for users
- 🔗 Easy URL-based customization
- 🌙 Theme support
- 📊 Rating, games played, win rate, and more



## 🖼️ Example

### Chess Profile Card

```markdown
![Chess Stats](https://chess-readme-stats.vercel.app/api?username=YOUR_USERNAME)
```

### Last Match Story

```markdown
![Last Match](https://chess-readme-stats.vercel.app/api/last-match?username=YOUR_USERNAME)
```



## 🚀 Quick Start

Replace `YOUR_USERNAME` with your Chess.com username.

```markdown
![Chess Stats](https://chess-readme-stats.vercel.app/api?username=hikaru)
```

Embed it anywhere that supports images:

- GitHub Profile README
- Personal Website
- Portfolio
- Blog
- Documentation
- Developer Profile Pages



## 🎯 Why Chess README Stats?

Developers proudly showcase:

- GitHub contributions
- LeetCode stats
- Codeforces ratings
- WakaTime activity

Now you can showcase your Chess.com journey too.

Whether you're a casual player, club competitor, or chess enthusiast, your profile can reflect your progress, consistency, and recent performance.
